# FAQ

## What is PM Architect AI?

PM Architect AI is a hosted AI product-management workspace that helps users turn rough ideas into structured PM artifacts such as PRDs, personas, user stories, technical architecture, and diagrams.

## Who is it for?

Primary audiences include:

- founders,
- product managers,
- builders and early product teams.

## Is it only an AI writing tool?

No. It combines:

- a discovery flow,
- generated PM artifacts,
- a persistent workspace with saved projects,
- integration settings for external knowledge workflows.

## Does it include a dashboard?

Yes. The live product includes an authenticated dashboard with saved projects and project detail views.

## Does it integrate with Synapse?

Yes. The product includes a Synapse integration flow for pushing generated artifacts into a broader knowledge system.

## Does this repository contain the source code?

No. This is a public product-information and support repository only.

## Is it self-hosted by end users?

No public self-hosted distribution flow is documented here.
