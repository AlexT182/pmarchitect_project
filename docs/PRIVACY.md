# Privacy Summary

PM Architect AI is a hosted web product.

High-level privacy expectations:

- public visitors may browse the landing page,
- authenticated users may access their dashboard and saved projects,
- project content and generated artifacts may be stored server-side to operate the service,
- configured third-party AI and integration providers may process content as part of the workflow.

This repository does not publish:

- application source code,
- private project workspaces,
- infrastructure secrets,
- user credentials,
- integration tokens.
