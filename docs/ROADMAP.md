# Public Roadmap

This is a lightweight public roadmap for PM Architect AI.

## Current product shape

- public landing page
- authenticated login flow
- dashboard with saved projects
- generated PM artifacts
- project detail workspace
- settings and Synapse integration

## Likely next improvements

- stronger artifact refinement loops
- richer collaboration and workflow controls
- smoother export and knowledge-sync paths
- better structure around project templates and repeatable workflows
- sharper onboarding for first-time users

## Scope note

This roadmap is directional only. It is not a release guarantee.
