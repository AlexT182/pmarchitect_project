# Security

## Reporting a vulnerability

If you believe you found a security issue affecting PM Architect AI, report it privately:

- Email: `taminhquan182@gmail.com`

Please include:

- a short description of the issue,
- reproduction steps if known,
- the affected URL or feature,
- screenshots or logs if they help,
- your contact information for follow-up.

## Please do not disclose publicly first

Do not open a public issue with:

- credentials,
- tokens,
- exploit details,
- user data,
- sensitive infrastructure information.

## Scope note

This repository is a public product-information repository. It does not contain the live application source code or private infrastructure.

Security reports should focus on:

- the live hosted product,
- public endpoints,
- authentication and authorization behavior,
- exposed data or configuration issues.
