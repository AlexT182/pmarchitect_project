# Support

PM Architect AI is a hosted product and is supported through direct contact.

## Fastest contact path

- Email: `taminhquan182@gmail.com`

## What to include when asking for help

Please send:

- the issue in one or two clear sentences,
- the page or flow where it happened,
- whether it affects login, dashboard, project generation, or settings,
- your account email if relevant,
- screenshots if the issue is visual,
- the browser or device if the issue is environment-specific.

## What this repo is for

This repository is for:

- product information,
- public FAQ,
- roadmap visibility,
- support routing.

This repository is not the source repository for the live product.

## Recommended support flow

### Public-site issues

Use email for:

- landing page issues,
- login problems,
- account questions,
- product-access questions.

### Workspace issues

Use direct email for:

- dashboard access issues,
- project-generation issues,
- artifact or export questions,
- Synapse integration questions,
- settings or account-profile issues.

## Do not send publicly

Do not post or share in public:

- account credentials,
- private project content,
- API keys or tokens,
- screenshots containing sensitive business information.

For anything sensitive, use email: `taminhquan182@gmail.com`.

## Response expectations

- Support is human-handled.
- Security- or privacy-sensitive reports should be sent privately.
